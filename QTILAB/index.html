8 Weeklong Online Internship Projects

CPU PerfMaster

CPU-PerfMaster is all about performing a detailed comparative analysis of two prominent Android smartphones by running CPU Bench Marks (BMs) to provide insights into their respective CPU capabilities through AUTOMATION using Python, Android Studio and ADB.

MOBIVIEW

MobiView is all about performing a detailed comparative analysis of two prominent Android smartphones by examining hardware and software specifications to provide insights into their respective strengths and weaknesses through AUTOMATION using Python, Android Studio and ADB

ShopAPK

"Android App Development with Kotlin" is specifically designed for engineering students and professionals to provide them with hands-on experience in Android Application development.

PyStab

PyStab is all about developing your own Python based automation framework for stability testing using Python, Android Studio and ADB.

GPU PerfMaster

GPU-PerfMaster is all about performing a detailed comparative analysis of two prominent Android smartphones by running GPU Bench Marks (BMs) to provide insights into their respective GPU capabilities through AUTOMATION using Python, Android Studio and ADB.

PySanity

PySanity is all about developing your own Python based automation framework for SANITY testing using Python, Android Studio and ADB.



CA with PowerBI

CA with PowerBI is all about performing a detailed comparative analysis of data using PowerBI tool. This is a good project to get exposure to PowerBI and build the career for Program Analyst and Manager role
